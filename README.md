# ManCape Central - Garage Bar & Man Cave Authority

A content-driven affiliate marketing website focused on garage bars, man caves, and home bar setups for the US market.

## 🎯 Project Overview

**Niche:** Garage Bar / Man Cave  
**Target Market:** US middle-class males (35-55 years old)  
**Revenue Model:** Affiliate marketing (Amazon Associates, ShareASale, Impact)  
**Traffic Sources:** Pinterest, Instagram, SEO  

## 🚀 Quick Start

### 1. Preview Locally
`ash
# Open index.html in your browser
double-click index.html
`

### 2. Deploy to GitHub Pages
`ash
# Initialize Git
git init
git add .
git commit -m "Initial commit"

# Create GitHub repo and push
git remote add origin https://github.com/YOUR_USERNAME/mancapecentral.git
git branch -M main
git push -u origin main

# Enable GitHub Pages in repo settings
`

### 3. Connect Domain (Optional)
`
1. Buy domain from Namecheap (~/year)
2. Add CNAME file with your domain
3. Configure DNS records in domain registrar
`
